# NTQ Forge

> Open-Source document generation engine for PRM, GSRP and technical publications.

NTQ Forge is a Python framework for generating structured technical documents from reusable components.

Instead of manually writing HTML, Markdown or PDF files, documents are built from semantic objects such as chapters, figures, tables and diagrams.

## Philosophy

NTQ Forge follows the same architectural principles as the PRM ecosystem.

- Clear Boundaries
- Reusable Components
- Renderer Independence
- Open Science
- Local First

## Planned Features

- HTML Renderer
- PDF Renderer
- Markdown Export
- SVG Diagram Engine
- NTQ Design Theme
- Vision Paper Generator
- PRM Paper Generator
- GSRP Paper Generator
